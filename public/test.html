Troubleshooting Steps:

  1. Make sure both frontend and backend servers are running
  2. Try clearing your browser cache (Ctrl+Shift+Delete)
  3. Try opening the app in an incognito/private window
  4. Check the browser console for any JavaScript errors (F12)
  5. Make sure you're accessing the correct URL: http://localhost:3011
